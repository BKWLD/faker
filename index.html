Faker Channel

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover maetl.github.com/faker
Listing available packages: 
pear remote-list -c faker
Installing a package: 
pear install faker/package_name
Installing a specific version/stability: 
pear install faker/package_name-1.0.0
pear install faker/package_name-beta
Receiving updates via a feed: 
http://maetl.github.com/faker/feed.xml
